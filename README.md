# 🌞 Summer products - E-commerce Website


A modern and responsive **Summer Products E-commerce Website** where users can explore seasonal products with a clean UI and smooth user experience. This project focuses on frontend functionality with authentication and protected routes.

## 🚀 Features

* 🛍️ Browse Summer Products
* 🔐 Authentication System (using Better Auth)
* 🔑 Google Sign-In Integration
* 🔒 Protected Routes (Only authenticated users can access certain pages)
* ⚡ Fast and responsive UI (built with modern technologies)
* 🎯 Clean and user-friendly design

---

## ⚠️ Project Limitations

* ❌ This is **NOT a full-stack project**
* ❌ No custom backend/database created
* ❌ Product data is static / JSON-based

## 🛠️ Technologies Used

* Next.js
* React

* Better Auth
